# Focus Flow 🚀

Focus Flow is a modern productivity web application that helps users stay focused, organized, and consistent with their daily goals.

🌐 Live Demo: https://focus-flow-bay-gamma.vercel.app/

---

✨ Features

  Management
- Add new tasks
- Edit and delete tasks
- Mark tasks as completed
- Organize daily work efficiently

 Calendar System
- Track tasks by date
- Manage daily schedules
- Easy calendar navigation

 Daily Streak Tracking
- Maintain productivity streaks
- Track consistency day by day


 Challenges
- Daily productivity challenges
- Weekly focus goals
- Motivates users to stay disciplined

  Habit Tracker
- Create healthy habits
- Track daily habit completion
- Build long-term routines
